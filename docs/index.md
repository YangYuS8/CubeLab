# CubeLab

CubeLab is an education-oriented platform built on top of `TencentCloud/CubeSandbox`.

It focuses on secure, isolated, one-learner-one-sandbox lab workflows for Linux, Shell, DevOps, and future AI Agent teaching scenarios.

## What This Repository Provides Today

- A monorepo baseline for long-term maintenance
- A documentation site with English and Simplified Chinese locales
- Draft package boundaries for sandbox abstraction and lab specification
- Collaboration rules for human and AI agent contributors
