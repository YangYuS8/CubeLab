# CubeLab

CubeLab 是一个基于 `TencentCloud/CubeSandbox` 的教育实验平台项目。

它面向 Linux、Shell、DevOps 以及未来的 AI Agent 教学场景，强调“一人一沙箱”的安全隔离实验流程。

## 当前仓库已提供

- 面向长期维护的 monorepo 基线
- 支持英文和简体中文的文档站点
- 沙箱抽象层与实验规范层的包边界草案
- 面向人类与 AI agents 的协作规则
